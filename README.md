# node-http-server

Simple static node server for testing 

## Installation
1. Install node ( requires V4+ )
2. run 

## Usage

   node server.js
   
   options: 
   
   --p rootPath  
   
       root directry to serve the content. Otherwise
   
       it will serve from current directry

   --noCache 
   
      It will serve static file with no cache control.
